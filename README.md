## 📋 사전 요구사항

- Docker DeskTop 설치
- Python 3.7+

## 🚀 빠른 시작

### 1. Clone
- git clone https://github.com/dangni02/vector-db-setup.git
- cd vector-db-setup

### 2. 실행
- chmod +x setup.sh
- ./setup.sh
